## 🚀 快速开始

### 环境要求

- Node.js >= 18
- npm / yarn / pnpm

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/Pitt-Lee/my-opensource-project.git

# 进入项目目录
cd my-opensource-project

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 📖 使用说明

# 构建生产版本
npm run build

# 运行生产服务器
